\subsection*{Data Availability}
To support reproducibility while respecting industrial confidentiality, we provide an anonymous artifact containing anonymized feature schemas, aggregate verification logs for all 3,343 tasks, scripts for computing verified properties within budget (VPB) and unsafe false negatives (UFN), and the scheduling and robustness analyses used in this study. Raw C/C++ source code and proprietary build configurations for System A cannot be disclosed. The artifact is available at: \url{https://anonymous.4open.science/r/VerifyOpt-XXXX}.
